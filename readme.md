Instructions to RUN THE FLASK BACKEND

1) create a .env file in the projects root directory with the values that are specified in the config.py file
2) type 'pip install -r requiremets.txt' in terminal , this will install the dependecies for the flask and database
3) make migrations with the flask migrate , refer -> https://flask-migrate.readthedocs.io/en/latest/
4) after migrations run the app.py file 
5) don't forget the set port the 9001 because the frontend is configured with default port of 9001
